Titanic Dataset - Exploratory Data Analysis (EDA)

Project Overview
This project focuses on performing Exploratory Data Analysis (EDA) on the Titanic dataset to understand the factors that influenced passenger survival. The analysis combines data visualization and feature engineering to uncover meaningful patterns and relationships within the data.

Objectives
Analyze survival patterns of passengers
Identify key factors affecting survival probability
Apply feature engineering to enhance analysis
Communicate insights using clear visualizations

Dataset
Dataset: Titanic passenger data
Includes features such as:
Passenger class
Age
Gender
Fare
Family size
Embarkation point

Data Cleaning & Preprocessing
Handled missing values (Age, Embarked, etc.)
Converted categorical variables into usable formats
Removed irrelevant or redundant columns
Ensured data consistency for analysis

Feature Engineering
Created Family Size feature
Extracted meaningful information from existing columns
Grouped and transformed variables for better insights

Data Visualization
Used visualizations to explore and communicate insights:
Survival rate by gender
Survival distribution across passenger classes
Age distribution of passengers
Fare vs survival patterns
Correlation between features

Key Insights
Gender played a major role in survival chances
Higher-class passengers had better survival rates
Family size influenced survival probability
Fare showed correlation with survival likelihood

Tools & Technologies
Python
Pandas
Matplotlib / Seaborn

Conclusion
This project highlights the importance of EDA and feature engineering in understanding datasets and building a strong foundation for predictive modeling.
